# AGENTS.md

## Project
This repository is for a local-first redesign of The Pawn Shop website.
Treat this as a simple informational business website unless told otherwise.

## Working rules
- Work only inside this repository.
- Do not add a Git remote.
- Do not run git fetch, git pull, git push, or any remote Git command unless the user explicitly says to do it.
- Do not use dangerous full-access or yolo modes.
- Prefer the simplest implementation that solves the problem cleanly.
- Keep dependencies minimal.
- Ask before adding or removing packages.
- Before coding, inspect the repo and state a short plan.
- After each meaningful completed task, run the relevant checks, summarize what changed, then create a local checkpoint commit.

## Git checkpoint rule
For local checkpoints, use:
git status
git add -A
git commit -m "PM checkpoint: <short description>"

## Quality bar
- Mobile-first and responsive
- Clear typography and hierarchy
- Fast-loading pages
- Basic accessibility
- No fake functionality
- No placeholder lorem ipsum unless explicitly requested

## Current phase
Local development only. No remote repo. No deployment yet.
